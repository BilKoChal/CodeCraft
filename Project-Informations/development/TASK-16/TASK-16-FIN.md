- **manifest:** Standalone display, dark theme colors, CodeCraft branding
- **workbox:** Precaches JS/CSS/HTML/ICO/PNG/SVG/WOFF2; CacheFirst for language chunks with 30-day expiry

### Service Worker Registration (main.tsx)
- Dynamic import of `virtual:pwa-register` for production-only inclusion
- `onNeedRefresh` callback — logs to console (Phase 1: toast notification)
- `onOfflineReady` callback — logs readiness
- `onRegisteredSW` — schedules hourly `registration.update()` checks
- `onRegisterError` — logs registration errors
- Graceful catch for environments without SW support

### Design Decisions
- SVG icons instead of PNG for simpler generation and infinite scalability
- `autoUpdate` strategy for zero-friction updates (no user prompt needed)
- Hourly background update checks for near-real-time deployment rollouts
