## Decisions Made

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Editor engine | CodeMirror 6 | 10–15x smaller than Monaco, mobile-first, tree-shakeable, 143 languages |
| State management | Zustand | 1 KB, persistent middleware, TypeScript-first |
| Storage | Dexie.js (IndexedDB) + LocalStorage | Structured data + quick prefs |
| Design identity | "Paper Desk" (Soft Industrial) | Code-first, warm amber accent, NOT VS Code clone |
| File navigator | "Sheet Stack" slide-over panel | Not a persistent sidebar — search-first, overlay |
| Theme count | 10 built-in themes | 5 dark, 4 light, 1 special (Terminal Green) |
| Code execution tiers | Tier 1 (JS/TS/HTML), Tier 2 (Python/C++), Tier 3 (Ruby/PHP), Tier 4 (syntax only) | Progressive complexity |
| License | MIT | Maximum adoption, community-friendly |
